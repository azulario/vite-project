# Projeto DIO - Calculadora e Login

Este projeto foi desenvolvido como parte do curso da DIO, utilizando React, TypeScript e Vite.

## Funcionalidades

- **Login**: Tela de login com validação simples e alerta de boas-vindas.
- **Calculadora**: Funcionalidades matemáticas básicas (soma, subtração, multiplicação).

## Tecnologias Utilizadas

- React
- TypeScript
- Vite
- Chakra UI
- Jest (para testes unitários)

## Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o projeto:
   ```bash
   npm run dev
   ```

3. Execute os testes:
   ```bash
   npm run test
   ```
